A interface File provê informações sobre arquivos e permite ao JavaScript  a acessar seu conteúdo.

São geralmente recuperados a partir de um objeto [FileList](https://www.todamateria.com.br/crase/) que é retornado como resultado da seleção, pelo usuário, de arquivos através do elemento [<input>](https://chatgpt.com/?utm_source=microsoft&utm_medium=paid_search&utm_campaign=MSFT_C_SEM_BBR_Core_CHT_BAU_ACQ_PER_MIX_ALL_LATAM_BR_PT_110625&c_id=570612907&c_agid=1183076501395658&c_crid=&c_kwid=kwd-73942727921680:loc-20&c_ims=&c_pms=293504&c_nw=o&c_dvc=c&msclkid=47d3580250c6102cd475321ab998c5b1), a partir do objeto [DataTransfer](https://www.booking.com/index.html?aid=1670475&label=ptbr-edge-ntp-topsites-curate-ana&chal_t=1780017527710&force_referer=https%3A%2F%2Fwww.bing.com%2Falink%2Flink%3Furl%3Dhttps%253A%252F%252Fwww.booking.com%252Findex.html%253Faid%253D1670475%2526label%253Dptbr-edge-ntp-topsites-curate-ana%26source%3Dmsn-ntp%26h%3D8p%252FpgycVUuBtDjfyoDc0OCs38LnSwdfqmjCT5L5%252Fvks%253D%26aff%3Dp%26p%3Dmkt%253Apt-br%253Bf%253AHHQEsLfFcI0bfkDE4F5cSxOwDTUAr8sDSse1rHdulh4%253D%253Btssls%253Awpod%26ocid%3Dmsedgntp%26pc%3DU531%26cvid%3D6a18e94b457a407db6ca4d035a94f947%26ei%3D8) utilizado em operações de arrastar e soltar, ou a partir da API `mozGetAsFile()` em um [HTMLCanvasElement](https://enem.inep.gov.br/participante/#!/). Em Gecko, códigos com privilégiios podem criar objetos File representando qualquer arquivo local sem a intereção do usuário (veja [Implementation notes](https://www.linkedin.com/home?utm_source=edge_ntp&utm_medium=top&utm_campaign=br&originalSubdomain=br) para mais informações.).
